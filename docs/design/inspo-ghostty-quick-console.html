速召台 — 把 Ghostty 的 Quick Terminal 移植进 Charter（研究稿 ①）

Ghostty 用户最高频的赞美之一是 Quick Terminal：一个全局热键，一个从屏幕顶滑下的持久终端，Esc 即走。 Charter 的对应痛点真实存在：在 Home / Task Room 里想跑一条命令，今天必须 ⌘E 切去编辑器、找底部坞、跑完再切回来 （FIX-HOME-TERMINAL 就是这个摩擦的补丁）。速召台把「跑一条命令」变成不离开当前上下文的动作， 并接上我们独有的两件事：工作上下文胶囊（终端可指向任务 worktree）与「发给 Room」（输出块变成任务上下文）。

借了什么、怎么落地 Ghostty 行为 → Charter 落法（不是像素级照搬）

Ghostty 原样用户为什么爱Charter 落法
Quick Terminal：全局热键从屏幕顶滑下一个持久终端，Esc / 失焦自动收回 「跑一条命令」不再需要切窗口/切工作区——上下文零丢失，会话还一直活着 速召台：⌥Space 覆在 Home / Room / 编辑器任何表面之上；PTY 持久；收回后焦点精确还原到召出前的位置。它不是第三个终端容器——就是坞里的一个 quick 会话，编辑器坞的 tab 列表里始终可见
新分屏/新 tab 继承当前 shell 的 cwd（shell integration） 终端永远开在「我正在干活的地方」，省掉每次 cd 升级为工作上下文胶囊：聚焦项目 / 最近项目 / 任务 worktree / Scratch 四选一，cwd 由主进程解析校验（对齐 terminal-parallel-vNext：渲染器给身份、主进程给路径）。指向 worktree 时挂 WORKTREE · 隔离 徽章——在隔离树里手工验证，不碰主树
OSC 133 语义块：命令输出可整块选中、复制 不用肉眼在滚动缓冲里数行 输出块可整选 + 「⤴ 发给当前 Room」：块变成回复里的引用附件（复用 @-ref chips 机制），终端输出第一次成为任务的一等上下文。块模型的完整故事见研究稿 ②
关闭可撤销（1.2 undo-close）：关闭其实先把终端隐藏保活一段可配置窗口（默认 5s），⌘Z 原样接回——因为进程从没死过；1.3 连快速终端状态都随会话恢复 手滑不再毁掉长会话 同一机制照搬：速召台/坞终端关闭先隐藏保活（默认 5s，可设），窗口内 ⌘Z 原样接回；超时才真正终止进程树并如实清账。含运行中前台进程的关闭确认不变（TERM-004）

三条设计原则

1 · 召之即来，挥之不改布局 NEW源自 Ghostty

速召台是覆层，不是布局的一部分：召出/收回不移动任何现有面板，Esc 或失焦（可设）即走， 焦点精确回到召出前的控件。与「检测只装饰、意图才搬家」（ADR-0017 rev.2）同一条肌理。

2 · 上下文是选出来的，不是猜出来的 NEW

Ghostty 继承 cwd；我们更进一步：胶囊显式声明这个会话属于哪棵树——聚焦项目、最近项目、 任务 worktree 或 Scratch。切换编辑器聚焦不杀速召台会话（终端上下文独立于编辑器聚焦， 对齐 vNext 方向稿）。

3 · 没有隐藏终端 KEEP

速召台的会话同时出现在编辑器坞的 tab 列表（⌥ quick）——同一 PTY 的两种呈现， 永远查得到、关得掉。TERM-001 的坞语义不变。

不变的部分（安全语义原样）

用户终端安全域 KEEP

速召台是用户终端（TERM-005）：手工命令不走 agent 审批，也永不冒充 agent 输出（TERM-006）。 「发给 Room」的输出是你提供的上下文，在时间线上署名 YOU。

worktree 记账边界 KEEP

指向任务 worktree 的速召台命令改了文件，走的是既有外部改动记账（watcher + 基线）， 在 Room 的改动清单里如实出现——不会被误记为 agent 的功劳。

TUI 键盘白名单 KEEP

速召台里跑全屏 TUI 时（vim / claude），Esc 属于 TUI；收回改用 ⌥Space 再按一次。 全局快捷键白名单问题沿用已记录的方向稿，不在本轮扩大。

刻意不搬的（以及为什么）

系统级全局热键（app 未聚焦也能召） SKIP · V1

Ghostty 的 quick terminal 是系统级的。Charter V1 只做应用内速召：跨应用召唤牵涉常驻代理与 辅助功能权限，收益/复杂度比不划算，留到桌面常驻形态一起议（需要 ADR）。

把速召台做成第二个 agent 入口 SKIP

诱惑存在：「顺手在速召台里问 agent」。不做——Charter 的 agent 入口是 Room 与 composer， 速召台保持纯终端。一个覆层两种心智（shell + 对话）会互相污染；想聊，⤴ 发给 Room 一步即达。
