方案 A · “+” 收进搜索行，下拉菜单

删掉底部两个突兀的按钮。搜索框右侧放一个 34px 方形 “+”（和 Sessions 面板里“搜索框 + 筛选按钮”是同一个版式语言），点击弹出下拉菜单：Open folder… / New project…。列表尾部与底部完全干净。

🖱 可交互：点击 “+” 打开 / 关闭菜单。

设计说明

  • 优点：底部彻底干净；“搜索框 + 方形按钮”与 Sessions 面板的筛选按钮完全同构，视觉语言统一；动作和“项目列表”在同一视线起点。
  • 优点：“+” 是行业通识（VS Code / Linear / Slack 的面板加号），无须学习。
  • 代价：动作从“一眼可见”变成“一次点击后可见”；所以空状态必须把两个入口直接铺在主体里（见③）。
  • 适合：加项目是低频动作、希望面板最安静的取向。
